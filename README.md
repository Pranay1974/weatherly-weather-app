## 📌 Overview

A modern and responsive weather application built using HTML, CSS, and JavaScript. It allows users to search for real-time weather updates of any city, displaying temperature, conditions, humidity, and wind speed with a visually engaging video background for a premium user experience.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Real-time temperature updates
* ☁️ Weather condition display (Cloudy, Sunny, Rainy, etc.)
* 💧 Humidity information
* 🌬️ Wind speed details
* 🎥 Dynamic video background
* ❌ Error handling for invalid cities

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & animations
* **JavaScript (Vanilla JS)** – Logic & API handling
* **Weather API** – Real-time data

## ⚙️ How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/modern-weather-app.git
   ```
2. Navigate to the project folder:

   ```bash
   cd modern-weather-app
   ```
3. Open `index.html` in your browser
